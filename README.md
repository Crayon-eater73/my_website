# my_website

This website is a multi‑page project showcasing several Blender creations inspired by items from the Resident Evil series.  
It includes reference images, finished renders, and organized pages for navigation, history, product details, and contact information.

The site is published using GitHub Pages at:  
https://crayon-eater73.github.io/my_website/

---

## Features

- Multi‑page website (Home, History, Products, Contact)
- Sticky navigation bar on all pages
- Responsive layout using Flexbox
- Scalable images with borders and hover styling
- Blender‑rendered assets displayed with reference images
- Clean typography using Google Fonts
- Gradient background and consistent color theme

---

## Pages Included

### **Home (`index.html`)**
- Introduction to the Blender project
- List of items created
- Reference images and finished product images

### **Products (`products.html`)**
- Individual product displays
- Images of final Blender renders

### **History (`history-index.html`)**
- Background information and context for the project

### **Contact (`about-us.html`)**
- Basic contact or “about the creator” information

---

## Blender Items Featured

- **Item Box**

- **Umbrella Corporation Logo**
- **Goop Juice**

Each item includes:
- A reference image
- A finished Blender render

## How to Run the Project Locally

1. Download or clone the repository.
2. Open the folder in VS Code.
3. Open `index.html` in your browser  
   **or**  
   Right‑click → *Open with Live Server*.
